# LiviaAI Studio

Bem-vindo ao LiviaAI Studio! 🚀

Esta é uma ferramenta de produtividade focada em marketing digital, desenvolvida para ajudar empreendedores a validar produtos e criar conteúdos de alta conversão para redes sociais.

## Funcionalidades
- **Validação de Produto:** Cálculo de margem, markup e score de oportunidade.
- **Copys TikTok:** Gerador de roteiros focados em retenção e vendas.
- **Direção de Cena:** Conversor de ideias em prompts técnicos para IAs.
- **Estúdio de Modelos e Vídeos:** Interface para simulação de criação de mídia com IA.

## Tecnologias
- HTML5
- CSS3 (Design Responsivo para Mobile)
- JavaScript (Lógica de processamento e UI)

## Deploy
Este projeto está configurado para deploy automático via Vercel.
